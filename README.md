# 🧬 USBFilesTheft.py - 深度隐藏的 USB 内容备份工具（仅限研究）

---

## ✅ 项目简介

`USBFilesTheft.py` 是一个基于 Windows 内核级别的深度隐藏文件备份工具，旨在持续监控所有连接的 USB 设备（包括未挂载的设备），并自动将数据复制到 软件目录\Copy 文件夹内该工具使用 `robocopy` 工具进行高效复制，并结合系统级隐藏、线程管理、设备变更监控等功能，实现高级潜伏行为。

> ⚠️ **重要警告：**
> 本工具仅为技术研究和学习目的提供。未经授权的使用可能构成**非法行为**，请严格遵守法律法规和用户许可，切勿用于非法目的。

## 📌 核心功能

### 🔒 唯一实例校验
- 检查全局互斥体 `Global\\WinSys_Final_v28`，确保只有一个实例运行。
- 防止冲突并避免重复执行。

### 🕵️‍♂️ 设备监控
- 通过 Windows `WM_DEVICECHANGE` 消息监听 USB 插入和拔出事件。
- 异步扫描所有 USB 驱动器（包括未挂载的）。

### 🧠 自动复制与清理
- 自动将所有 USB 设备中的文件复制到设定的目标目录。
- 每次复制后自动清理目标目录空间，避免磁盘写满导致系统异常。
- 使用 `robocopy` 进行高效复制，支持多线程模式以加快传输。
- 当软件在可移动磁盘下时，不会有冲突和死循环

### 🕵️‍♀️ 隐藏机制
- 为目标目录和自身文件设置**隐藏 + 系统 + 临时**属性。
- 使用 `SetFileAttributesW` 函数隐藏文件，避免系统扫描和触发安全报警。
- 对自身进程进行隐藏，并锁定到特定 CPU 核心以避免被检测。

---

## 📦 目录结构

### `TARGET_ROOT`
- 自动创建在当前脚本所在目录下的 `Copy` 文件夹。
- 所有复制内容将被隐藏在这个目录中。
- 每个 USB 设备会创建一个临时目录并以格式 `S_<SN>_<Timestamp>` 命名。

### 隐藏机制
- 目标目录属性：**隐藏 + 系统 + 临时**
- 进程隐藏：使用系统级窗口隐藏、优先级调整等手段。

## 📚 技术原理

### 1. **Windows 驱动监控**
- 使用 `WM_DEVICECHANGE` 消息捕获 USB 插入 / 拔出事件。
- 通过注册自定义窗口类来实现**低调监听**，避免被系统监控到。

### 2. **多线程高速复制**
- 使用 `robocopy` 配合多线程参数 `/MT` 加速复制。
- 跳过目录比较减少系统日志，通过 `/LOG:NUL` 关闭输出。
- `/XJ` 排除只读文件，`/SL` 仅复制根目录。

### 3. **进程隐藏与优化**
- 设置进程运行在**高优先级**，避免被系统终止。
- 锁定到**最后一个 CPU 核心**，减少系统感知。
- 配合系统级隐藏方式，提升隐蔽性。

## 📌 进程行为分析（仅限研究）

| 行为 | 描述 |
|------|------|
| `CreateMutex` | 保证唯一实例运行 |
| `SetFileAttributesW` | 设置文件为隐藏 / 系统 / 临时属性 |
| `SetPriorityClass` | 设置进程为高优先级 |
| `SetProcessAffinityMask` | 锁定到特定 CPU 核心 |
| `robocopy` | 使用高级复制命令进行大量文件搬运 |
| `WM_DEVICECHANGE` | 监控 USB 插入和拔出事件 |
